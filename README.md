# ChatHub Backend

This is the backend repository for **ChatHub**, a real-time chat application built with **Node.js** and **Express.js**. It provides APIs for real-time messaging, voice/video calls, notifications, user management, and more.

---

## Features

- **Real-Time Chat**: Send and receive messages instantly.
- **Real-Time Notifications**: Push notifications for messages and calls.
